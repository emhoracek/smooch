Smooch

Smooch is an implementation of the Kisekae Set System for the web. You can learn more from the Github repo.

Demo sets

Eventually Smooch will be a web app that allows anyone to view any set. For now I'm just running it on my machine and generating the JavaScript, HTML, and images. Whenever I get a set working, I'll upload it here.

  • Arwen by Aragonite and Anime Craze (2003) (on Otakuworld)
  • Aurora by Punky (2001) (on Otakuworld) -- this doll has snap-to and animation effects in the original which aren't reproduced here
  • Hyacinth Delilah by Dorothy (2001) (on Otakuworld)
  • Little Washu by Alison (2001) (on Otakuworld)
  • Mini-Skuld by MUX (1993) (on Otakuworld)
  • Sakasagami no Yura by Kimiki (2000) (on Otakuworld)
